Margin, padding, border

Margin, padding and border are CSS styling components that together make up the foundation of layout on the web. HTML elements are essentially little rectangles and within these rectangles are four layers; margin, border, padding and content. The image below gives you a visual perspective of what an element looks like in a box model (a box model is the technical term used for the little rectangle).

Padding definition: Padding refers to the inner margin of a CSS box - between the outer edge of the content layer and the inner edge of the border layer. The size of the padding can be adjusted on all four sides at once, or on one side at a time.

Border definition: The border of a CSS box sits between the outer edge of the padding and the inner edge of the margin. By default, the border is individual but you can set the thickness, style and color of the border to make it appear. The border has many different styles and some examples of these styles are ‘solid’, ‘dotted’, ‘grooved’ and ‘ridged’. Added to this you can customise your border to have more of a curvature or to be more box shaped.

Margin definition: The margin surrounds a CSS box, and pushes up against other CSS boxes in the layout. It works the same as padding and is used for spacing; the margin size can be adjusted using the individual properties of margin-top, margin-right, margin-bottom, and margin-left.

The difference: Firstly, the difference between padding and margin is basically where they’re located. They are both used for spacing. Padding spaces content from the border, whereas the margin spaces the element from other elements. The border differs a lot from margin and padding as it isn’t necessarily used for spacing. Instead it is used to frame each element and offers a more stylish look.
